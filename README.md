# qwack 🦆

**Your sassy duck prompt quality coach.** Automatically roasts bad prompts and fixes them for you.

---

## What Qwack Does

- **Prompt Sentry** — Watches every prompt and sassily critiques bad ones (below 7/10) with specific improvements
- **AI Coach** — Teaches you how to prompt effectively for any task, with ready-to-use examples
- **Skill Translator** — Explains any skill in plain English with real-world examples
- **Pattern Spotter** — Notices repetitive workflows and suggests creating skills to automate them

---

## Quick Start

```
/qwack-on          Turn on prompt sentry
/qwack-off         Turn it off
/qwack-plan task   Learn how to prompt for anything
/qwack-explain     Understand any skill
```

---

## Why Qwack?

Bad prompts waste time. Qwack teaches you to prompt better by example — not lectures, but quacks.

You type: "fix my code"

Qwack replies: "🦆 **QUACK!** That's a 3/10. Try: 'Fix the authentication bug in src/components/Login.js — users get Invalid token error after password reset.'"

You learn. Next time, you prompt better.
